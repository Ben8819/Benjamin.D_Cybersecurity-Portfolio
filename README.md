# Ben’s Cybersecurity Portfolio

This repository documents my journey transitioning from IT support to cybersecurity, with a focus on SOC Analyst skills.

The goal is to track:
- What I learn
- The labs I complete
- The tools I practice
- The projects I build over time

This portfolio is a work in progress and will grow as I continue learning.

## 📁 Structure

### /learning  
Notes, summaries, and concepts I study.

### /labs  
Documented hands-on labs (Wireshark, Splunk, Sysmon, Linux logs, GNS3, etc.)

### /projects  
Longer projects like a home SOC lab, Wazuh, scripting, automation, etc.
